# Corona-Tests

## Description
A website that allowsadding a corona patient to database and to update, delete and view his details.
## Built with:
### Client
-React
-React Router 
-Reduxjs/toolkit
-Rtk Query
-Bootstrap

### Server
-NodeJs
-Express
-joi

### Database
-MySQL

## Getting started
### Installing
run `npm install`
run `npm start` on server and then on client side.
The localhost will then pop up in the browser.
